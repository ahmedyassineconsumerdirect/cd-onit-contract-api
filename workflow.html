Create Contract

Onit API
ⓘ
Looks up the Other Party Contact ID by CONTACT_EMAIL, then creates the CLM contract. The contact must already exist in Onit.
GET /api/atom_dictionaries/5ce59de8d8a1be14f7002d9a/atoms.json?filter=[encoded]
Filter Parameter (URL-encoded JSON) 
[{
  "field": "p_email",
  "type": "text",
  "comparison": "equals",
  "value": "ahmed@consumerdirect.com"  // CONTACT_EMAIL from CSV
}]
Response 
{
  "success": true,
  "data": [{
    "name": "Ahmed Yassine",
    "_id": "d09c7aee-de56-4976-9327-2cd35669a543"  // ← contact_id
  }]
}

Send for Signature

Onit API
ⓘ
Finds the document by searching the Documents app (5c6c79b19fcae642af000086) where p_parent_id matches the contract atom ID, then creates a Send for Signature atom with 2 signers. HelloSign sends the signature request automatically.
GET /api/atom_dictionaries/5c6c79b19fcae642af000086/atoms.json?filter=[encoded]
Filter: p_parent_id = contract atom_id 
[{
  "field": "p_parent_id",
  "type": "text",
  "comparison": "equals",
  "value": "6294b97f-f975-4a35-8f0d-bbaefd03ee3b"
}]
Response (pick the "Order Form" document) 
{
  "success": true,
  "data": [
    { "name": "Consumer Direct - Order Form.docx", "_id": "b2d4d635-..." },  // ← use this
    { "name": "Document 1029645", "_id": "..." }  // auto-generated, skip
  ]
}
